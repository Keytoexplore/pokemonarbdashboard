# Setup (S12a only)

This project is configured to track **S12a (VSTAR Universe)** only.

## Install

```bash
npm install
```

## Generate data/prices.json

```bash
npm run build:s12a
```

To force refresh and ignore caches:

```bash
npm run build:s12a:force
```

## Run the dashboard

```bash
npm run dev
```
